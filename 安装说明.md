# HCI IDE MVP · 安装与运行说明

这是一个完整的离线打包，已内置 Node 运行时和依赖。请根据你的电脑类型选择对应方式。

---

## 方式一：同款 Apple 芯片 Mac（M1/M2/M3...）— 开箱即用，无需联网

1. 解压本压缩包
2. 打开「终端」，进入解压后的文件夹
3. 运行：

   ```bash
   ./start.sh
   ```

4. 浏览器打开 http://localhost:5173/

> 如果提示权限不足，先执行 `chmod +x start.sh` 再运行。

---

## 方式二：Intel Mac / Windows / 其他平台 — 需要本机有 Node（18+）并联网

包内的 `node_modules` 和 `.node` 是 Apple 芯片 Mac 专用的，换平台需要重装依赖：

1. 删除 `node_modules` 和 `.node` 两个文件夹
2. 确保电脑已安装 Node.js 18 或以上（https://nodejs.org）
3. 在项目文件夹中运行：

   ```bash
   npm install
   npm run dev
   ```

4. 浏览器打开终端提示的地址（通常是 http://localhost:5173/）

---

## 方式三：只想看演示，不想装环境 — 用浏览器打开静态网站

包内 `dist/` 目录是已构建好的静态网站。用任意静态服务器托管即可，例如：

```bash
# 如果有 Python
cd dist && python3 -m http.server 8080
# 然后浏览器打开 http://localhost:8080/
```

> 注意：直接双击 `dist/index.html` 可能因浏览器安全策略无法加载资源，建议用上面的本地服务器方式。
